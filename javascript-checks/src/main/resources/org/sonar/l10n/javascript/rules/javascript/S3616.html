逗号运算符(，)从左到右计算其操作数，并返回第二个操作数。这在某些情况下很有用，但是在switch case中就错了。在这种情况下，您可能认为您正在紧凑地处理多个值，但是只会处理逗号列表中的最后一个值。剩下的将会变成默认值。

类似地，逻辑或运算符(||)将不能在switch case)中工作，在执行时只考虑第一个参数。

不合规的代码示例

switch a {
  case 1,2:  // 不合规; 这个案例只处理过2个
    doTheThing(a);
  case 3 || 4: // 不合规; 只处理“3”
    doThatThing(a);
  case 5:
    doTheOtherThing(a);
  default:
    console.log("Neener, neener!");  // 这发生在a==1或a== 4时
}

合规的解决方案

switch a {
  case 1:
  case 2:
    doTheThing(a);
  case 3:
  case 4:
    doThatThing(a);
  case 5:
    doTheOtherThing(a);
  default:
    console.log("Neener, neener!");
}
