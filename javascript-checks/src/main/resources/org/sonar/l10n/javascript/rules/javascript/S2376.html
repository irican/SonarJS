当使用属性的setter创建对象而没有属性的getter时，属性是不可访问的，因此是无用的。

此规则还强制执行相反的情况(getter但没有setter)。

不合规的代码示例

var obj = {
    set foo(value) {
        this.fooval = value;
    }
};

合规的解决方案

var obj = {
    set foo(value) {
        this.fooval = value;
    },
    get foo() {
        return this.fooval;
    }
};

or 

var obj = {
    setFoo(value) {    // 标准方法，而不是setter
        this.fooval = value;
    }
};
