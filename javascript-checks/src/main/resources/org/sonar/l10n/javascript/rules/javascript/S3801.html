与强类型语言不同，JavaScript不强制函数返回类型。这意味着函数中的不同路径可以返回不同类型的值，这可能会让用户非常困惑，而且维护起来也非常困难。

特别是一个函数，在JavaScript中，在以下任何一种情况下都会返回 undefined ：

  • 它退出时没有返回 return 语句。
  • 它执行一个没有值的 return 。

此规则验证是否始终或从不为函数中的每个路径指定返回值。

不合规的代码示例

function foo(a) { // 不合规, 函数没有返回就退出
  if (a == 1) {
    return true;
  }
}

合规的解决方案

function foo(a) {
  if (a == 1) {
    return true;
  }
  return false;
}
