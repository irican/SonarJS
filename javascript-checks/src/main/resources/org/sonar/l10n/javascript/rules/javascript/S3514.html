ECMAScript 2015引入了同时从对象或数组中提取和分配多个数据点的功能。这被称为“析构”，它允许您压缩样板代码，这样您就可以专注于逻辑。

当从同一个对象或数组中提取多个数据片段并将其分配给变量时，这条规则会引发一个问题。

不合规的代码示例

function foo (obj1, obj2, array) {
  var a = obj1.a;  // 不合规
  var b = obj1.b;

  var name = obj2.name;  // 忽略;只有一个提取和分配

  var zero = array[0];  // 不合规
  var one = array[1];
}

Compliant Solution

function foo (obj1, obj2, array) {
  var {a, b} = obj1;

  var {name} = obj2;  // 这种语法之所以有效，是因为var名称和属性名称是相同的

  var [zero, one] = array;
}
