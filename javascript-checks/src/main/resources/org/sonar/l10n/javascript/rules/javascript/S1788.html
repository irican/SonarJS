为函数参数定义默认值的能力可以使函数更容易使用。默认参数值允许调用者指定任意多或少的参数，同时获得相同的功能并最小化样板文件和包装器代码。

但是所有带有默认值的函数参数都应该在没有默认值的函数参数之后声明。 否则，调用者就不可能利用默认值;它们必须重新指定默认值，或者传递undefined，以便 "get to" 非默认参数。

不合规的代码示例

function multiply(a = 1, b) {  // 不合规
  return a*b;
}

var x = multiply(42);  // 返回未定义的NaN

合规的解决方案

function multiply(b, a = 1) {
  return a*b;
}

var x = multiply(42);  // 预期返回42
