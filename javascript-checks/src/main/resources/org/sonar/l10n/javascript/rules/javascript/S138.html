一个函数增长得太大，往往会聚集太多的职责。

这些函数不可避免地变得更难理解，因此也更难维护。

在特定阈值以上，强烈建议重构为更小的函数，这些函数专注于定义良好的任务。

这些较小的函数不仅更容易理解，而且可能更容易测试。

例外

此函数忽略立即调用的函数表达式(IIFE)，这些函数是在没有分配名称的情况下创建和调用的。

(function () { // 被这条规则忽略

  function open() {  // 经典函数声明;不忽略
    // ...
  }

  function read() {
    // ...
  }

  function readlines() {
    // ...
  }
})();
