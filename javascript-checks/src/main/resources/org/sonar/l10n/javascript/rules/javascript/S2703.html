JavaScript变量范围可能特别难以理解和正确。当您考虑 accidental 创建全局变量时，情况会变得更糟，这是在函数中声明变量或在for 子句，不使用let, const 或 var 关键字。

ECMAScript 2015引入了 let 和 const，现在是变量声明的首选关键字。

不合规的代码示例

function f(){
  i = 1;         // 不合规; i 是全局的

  for (j = 0; j < array.length; j++) {  // 不合规; j 现在也是全局的
    // ...
  }
}

合规的解决方案

function f(){
  var i = 1;

  for (let j = 0; j < array.length; j++) {
    // ...
  }
}
