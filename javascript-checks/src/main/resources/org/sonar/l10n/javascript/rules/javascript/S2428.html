对象文本语法，它在对象声明中初始化对象的属性比另一种方法更干净、更清晰:创建一个空对象，然后逐个给它属性。

当遇到以下模式时，就会出现问题:

  • 创建一个空对象。
  • 连续的单行语句向创建的对象添加属性。

注意，此规则要求Node.js在分析期间可用。

不合规的代码示例

let person = {};  // 不合规
person.firstName = "John";
person.middleInitial = "Q";
person.lastName = "Public";

合规的解决方案

let person = {
  firstName: "John",
  middleInitial: "Q",
  lastName: "Public",
}
