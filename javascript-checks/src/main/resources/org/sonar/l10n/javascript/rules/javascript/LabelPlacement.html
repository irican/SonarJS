任何语句或语句块都可以用标签标识，但是这些标签只能用于 while, do-while 和 for 语句。 在任何其他上下文中使用标签都会导致非结构化、混乱的代码。

不合规的代码示例

myLabel:if (i % 2 == 0) {  // 不合规
  if (i == 12) {
    print("12");
    break myLabel;
  }
  print("Odd number, but not 12");
}

合规的解决方案 

myLabel:for (i = 0; i < 10; i++) {   // 合规
  print("Loop");
  break myLabel;
}
