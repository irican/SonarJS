在一行代码块中可以省略大括号，例如， if 语句或for 循环，但这样做可能会误导和诱导错误。

当一行代码块后的行缩进表示有意将这些行包含在代码块中，但是没有大括号意味着这些行将无条件地执行一次时，这条规则就会产生问题。

不合规的代码示例

if (condition)
  firstActionInBlock();
  secondAction();  // 不合规; 无条件执行
thirdAction();

String str = null;
for (int i = 0; i < array.length; i++)
  str = array[i];
  doTheThing(str);  // 不合规; 仅在最后一个数组元素上执行

合规的解决方案

if (condition) {
  firstActionInBlock();
  secondAction();
}
thirdAction();

String str = null;
for (int i = 0; i < array.length; i++) {
  str = array[i];
  doTheThing(str);
}

请参阅

  • MITRE, CWE-483 - Incorrect Block Delimitation
  • CERT, EXP52-J. - Use braces for the body of an if, for, or while statement
