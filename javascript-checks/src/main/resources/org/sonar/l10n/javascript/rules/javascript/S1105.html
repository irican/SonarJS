共享命名约定允许团队有效地协作。当没有将左大括号放在一行代码的末尾时，此规则会引发一个问题。

不合规的代码示例

if (condition)
{                                                      //不合规
  doSomething();
}

合规的解决方案

if (condition) {                                   //合规
  doSomething();
}

例外

作为参数出现的对象字面量可以从它们自己的行开始。

functionWithObject(
   {                                                 //合规
        g: "someValue"
   }
);
