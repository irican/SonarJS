当两个函数具有相同的实现时，要么是一个错误(有其他意图)，要么是故意复制，但是可能会使维护人员感到困惑。在后一种情况下，应该重构代码。

注意，此规则要求Node.js在分析期间可用。

不合规的代码示例

function calculateCode() {
  doTheThing();
  doOtherThing();
  return code;
}

function getName() {  // 不合规
  doTheThing();
  doOtherThing();
  return code;
}

合规的解决方案

function calculateCode() {
  doTheThing();
  doOtherThing();
  return code;
}

function getName() {
  return calculateCode();
}

例外

小于3行的函数将被忽略。
