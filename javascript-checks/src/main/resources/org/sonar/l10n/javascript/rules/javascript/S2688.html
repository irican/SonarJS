相反，查看变量是否等于NaN的最佳方法是使用Number.isNaN()，自ES2015年以来，或者(可能与直觉相反)将其与自身进行比较。 由于 NaN !== NaN, 当 a !== a时，您知道它必须等于NaN。

不合规的代码示例

var a = NaN;

if (a === NaN) {  // 不合规; 总是错误的
  console.log("a is not a number");  // 这是死代码
}
if (a !== NaN) { // 不合规; 总是正确的
  console.log("a is not NaN"); // 这种说法不一定正确
}

合规的解决方案

if (Number.isNaN(a)) {
  console.log("a is not a number");
}
if (!Number.isNaN(a)) {
  console.log("a is not NaN");
}
