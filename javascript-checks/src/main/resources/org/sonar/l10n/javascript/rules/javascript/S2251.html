具有永远无法达到的停止条件的for 循环，例如计数器移动方向错误的for循环，将无限运行。虽然有时需要无限循环，但惯例是构造 while 循环之类的循环。更典型的情况是， 无限 for 循环是一个bug。

不合规的代码示例

for (var i = 0; i < strings.length; i--) { // 不合规;
  //...
}

合规的解决方案

for (var i = 0; i < strings.length; i++) {
  //...
}

请参阅

  • CERT, MSC54-J. - Avoid inadvertent wrapping of loop counters
