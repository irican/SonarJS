当函数调用中参数的名称与函数参数的名称匹配时，它有助于代码更清晰、更可读。但是，当名称匹配时，但是以不同于函数参数的顺序传递时，它指出参数顺序中的错误，这可能会导致意外的结果。

不合规的代码示例

function divide(divisor, dividend) {
  return divisor/dividend;
}

function doTheThing() {
  var divisor = 15;
  var dividend = 5;

  var result = divide(dividend, divisor);  // 不合规；操作成功，但结果出乎意料
  //...
}

合规的解决方案

function divide(divisor, dividend) {
  return divisor/dividend;
}

function doTheThing() {
  var divisor = 15;
  var dividend = 5;

  var result = divide(divisor, dividend);
  //...
}
