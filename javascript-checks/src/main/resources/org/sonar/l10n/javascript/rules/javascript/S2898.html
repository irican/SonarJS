虽然:<element_type> 和 [type="<element_type>"] 都可以在jQuery中使用它们的类型来选择元素，但是,[type="<element_type>"] 要快得多，因为它可以利用现代浏览器中的原生DOM querySelectorAll() 方法

This rule raises an issue when following selectors are used:

  • :checkbox
  • :file
  • :image
  • :password
  • :radio
  • :reset
  • :text

不合规的代码示例

var input = $( "form input:radio" ); // 不合规

合规的解决方案 

var input = $( "form input[type=radio]" ); // 合规
