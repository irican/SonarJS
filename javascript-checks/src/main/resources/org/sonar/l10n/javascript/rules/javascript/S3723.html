现代浏览器会忽略不需要的逗号，所以除非支持IE 8应用程序，否则使用逗号没有什么不好的。由于它们简化了添加新属性的过程，所以最好使用它们。当对象声明或数组声明中的最后一项不以逗号结尾，也不与右大括号或大括号位于同一行时，此规则会引发问题。

不合规的代码示例

var joe = {
  fname: "Joe",
  lname: "Smith"      // 不合规
};

Compliant Solution

var joe = {
  fname: "Joe",
  lname: "Smith",    // OK
};

var joe = {
  fname: "Joe",
  lname: "Smith"};   // OK
