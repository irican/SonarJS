一元运算符+和-可用于将某些值类型转换为数值。但并不是每个值都可以转换为Number type;将其与对象一起使用，结果将是NaN(不是数字)。这可能会使维护人员感到困惑。

不合规的代码示例

var obj = {x : 1};
doSomethingWithNumber(+obj);    // 不合规

function foo(){
  return 1;
}
doSomethingWithNumber(-foo);    // 不合规

合规的解决方案

var obj = {x : 1};
doSomethingWithNumber(+obj.x);

function foo(){
  return 1;
}
doSomethingWithNumber(-foo());

var str = '42';
doSomethingWithNumber(+str);

例外

一元+ 和 - c可与对应原始类型的对象使用, + 可与 Date使用.

var b = new Boolean(true);
doSomethingWithNumber(-b);  // 合规
var timestamp = +new Date();  // 合规
