全局作用域中声明的任何变量或函数都会隐式地附加到全局对象 (浏览器环境中的window 对象). 要使其显式，这个变量或函数应该是 window属性。 当仅在本地使用它时，应该使用const 或 let 关键字声明它(自ECMAScript 2015以来)，或者在一个立即调用的函数表达式(IIFE)中声明它。

使用模块时不应激活此规则。

不合规的代码示例

var myVar = 42;       // 不合规
function myFunc() { } // 不合规

合规的解决方案

window.myVar = 42;
window.myFunc = function() { };

或

let myVar = 42;
let myFunc = function() { }

或 

// IIFE
(function() {
  var myVar = 42;
  function myFunc() { }
})();
