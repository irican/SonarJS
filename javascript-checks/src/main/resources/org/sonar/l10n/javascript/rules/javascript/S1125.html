为了提高代码可读性，在比较表达式 == 和 != 中应该避免布尔值。

此规则还报告冗余布尔操作。

注意，此规则要求Node.js在分析期间可用。

不合规的代码示例

let someValue = "0";
// ...

if (someValue == true) { /* ... */ }
if (someBooleanValue != true) { /* ... */ }
doSomething(!false);

合规的解决方案

if (someValue && someValue != "0") { /* ... */ }
if (!someBooleanValue) { /* ... */ }
doSomething(true);
