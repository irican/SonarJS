没有理由导入您不使用的模块;而每一个不这样做的理由都是:这样做毫无必要地增加了负担。

最后，两次导入一个模块是没有意义和令人困惑的。

不合规的代码示例

import A from 'a';      // 不合规，A没有被使用
import { B1 } from 'b';

console.log("My first JavaScript...");

import { B1 } from 'b'; // 不合规，已经被导入

console.log(B1);

合规的解决方案

import { B1 } from 'b';

console.log("My first JavaScript...");

console.log(B1);
