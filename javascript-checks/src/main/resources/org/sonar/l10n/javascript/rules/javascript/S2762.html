jQuery不会为您缓存元素。如果您已经进行了可能需要再次进行的选择，则应该将选择保存到变量中，而不是重复进行选择。

不合规的代码示例

默认阈值为2

$( "p" ).hide();
$( "p" ).show();  // 不合规

合规的解决方案

var paragraph = $( "p" );

paragraph.hide();
paragraph.show();

例外

当DOM更改时，不会更新存储的选择。因为变量可能需要更新，所以这个规则忽略了在赋值过程中重复的选择。

var paragraph = $("p");

// ...

paragraph = $("p");
