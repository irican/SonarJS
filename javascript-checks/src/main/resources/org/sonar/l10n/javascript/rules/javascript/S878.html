逗号操作符接受两个表达式，从左到右执行它们，并返回第二个表达式的结果。 使用这个操作符通常不利于代码的可读性和可靠性，通过其他方法也可以达到同样的效果。

不合规的代码示例

i = a += 2, a + b;  // i的值是多少?

合规的解决方案

a +=  2;
i = a + b;

例外

在 for 循环的初始化和递增表达式中，允许使用逗号运算符。

for(i = 0, j = 5; i < 6; i++, j++) { ... }
