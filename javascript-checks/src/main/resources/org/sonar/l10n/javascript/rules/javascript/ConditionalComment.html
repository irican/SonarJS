Internet Explorer提供了一种在运行时使用条件注释(由注释中找到的@cc_on语句激活)更改JavaScript代码的方法。 使用这种预处理特性会降低可读性和可维护性，并可能阻碍自动化工具。更重要的是，它是特定于Internet Explorer的，不能用于其他浏览器。

大多数情况下，使用这些条件注释可以通过一些重构轻松避免——使用现代跨浏览器JavaScript框架和库。

不合规的代码示例

/*@cc_on
  @if (@_jscript_version >= 5.5)
    document.write("You are using IE5.5 or newer");
  @else
    document.write("You are using IE5 or older");
  @end
  @*/
