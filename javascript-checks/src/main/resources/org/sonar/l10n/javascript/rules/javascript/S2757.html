运算符对的使用(=+, =- 或 =!) 将编译和运行反向的单个操作符 (+=, -= 或 !=)，但不会产生预期的结果。

当 =+, =- 和 =! 在两个操作符之间没有任何空格，并且当后面至少有一个空格时使用。

不合规的代码示例

let target =-5;
let num = 3;

target =- num;  // 不合规; target = -3.这是真的吗?
target =+ num; //不合规; target = 3

合规的解决方案

let target = -5;
let num = 3;

target = -num;  // 合规; 分配num逆值的意图很明显
target += num;
