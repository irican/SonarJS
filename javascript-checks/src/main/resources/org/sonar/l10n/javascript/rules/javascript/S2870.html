可以使用delete操作符从任何对象中删除属性。数组是对象，所以这里也可以使用delete操作符，但是如果是，数组中就会留下一个洞，因为索引/键不会移动来反映删除。

删除某一指数项的适当方法如下:

  • Array.prototype.splice - add/remove elements from the array
  • Array.prototype.pop - add/remove elements from the end of the array
  • Array.prototype.shift - add/remove elements from the beginning of the array

不合规的代码示例

var myArray = ['a', 'b', 'c', 'd'];

delete myArray[2];  // 不合规. myArray => ['a', 'b', undefined, 'd']
console.log(myArray[2]); // 期望值为“d”，但输出未定义

合规的解决方案

var myArray = ['a', 'b', 'c', 'd'];

// removes 1 element from index 2
removed = myArray.splice(2, 1);  // myArray => ['a', 'b', 'd']
console.log(myArray[2]); // 输出 'd'
