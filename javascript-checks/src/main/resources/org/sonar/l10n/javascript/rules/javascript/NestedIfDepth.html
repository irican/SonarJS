嵌套的 if, for, while, switch, and try 语句是生成所谓 "Spaghetti code"的关键成分。

这样的代码很难读取、重构和维护。

不合规的代码示例

默认阈值为3:

  if (condition1) {                  // 合规 - depth = 1
    /* ... */
    if (condition2) {                // 合规 - depth = 2
      /* ... */
      for(let i = 0; i < 10; i++) {  // 合规 - depth = 3, 不超过限额
        /* ... */
        if (condition4) {            // 不合规 - depth = 4
          if (condition5) {          // Depth = 5, 超出限制，但只有depth = 4 时才报告问题
            /* ... */
          }
          return;
        }
      }
    }
  }
