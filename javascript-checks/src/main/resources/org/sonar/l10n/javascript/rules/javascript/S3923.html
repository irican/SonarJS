在switch 或 if 链中具有相同实现的所有分支是错误的。 要么复制粘贴错误，应该执行一些不同的操作，要么根本不应该有 switch/if 链。 注意，此规则不适用于没有 else-s 的if 链,也不适用于没有default子句的switch-es 。

注意，此规则要求Node.js在分析期间可用。

不合规的代码示例

if (b == 0) {  // 不合规
  doOneMoreThing();
}
else {
  doOneMoreThing();
}

let a = (b == 0) ? getValue() : getValue();   // 不合规

switch (i) {  // 不合规
  case 1:
    doSomething();
    break;
  case 2:
    doSomething();
    break;
  case 3:
    doSomething();
    break;
  default:
    doSomething();
}
