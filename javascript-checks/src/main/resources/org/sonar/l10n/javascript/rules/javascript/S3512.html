ECMAScript 2015增加了使用模板文字而不是连接的功能。因为它们的使用更清晰、更简洁，所以在支持ECMAScript 2015的环境中更受欢迎。

当由两个或多个连接的结果创建字符串时，此规则会引发问题。

不合规的代码示例

function sayHello(name) {
  console.log("hello " + name);  // 忽略
}

function madLib(verb, noun) {
  console.log("I really " + verb + " one or two " + noun);  // 不合规
}

合规的解决方案

function sayHello(name) {
  console.log(`hello ${name}`);  // 之前没有提出过任何问题，但现在好多了
}

function madLib(verb, noun) {
  console.log(`I really ${verb} one or two ${noun}`);
}
