重写一个对象会改变其行为，并可能影响使用该对象的所有代码。因此，覆盖标准的内置对象可能会对以前工作的代码产生广泛的、潜在的灾难性影响。

此规则检测下列本机对象的覆盖:

  • 基本对象—— Object, Function, Boolean, Symbol, Error, EvalError, InternalError, RangeError, ReferenceError, SyntaxError, TypeError, URIError
  • 数字和日期 - Number, Math, Date
  • 文本处理 - String, RegExp
  • 索引集合 - Array, Int8Array, Uint8Array, Uint8ClampedArray, Int16Array, Unit16Array, Int32Array, Uint32Array, Float32Array, Float64Array
  • 键集合 - Map, Set, WeakMap, WeakSet
  • 结构化数据 - ArrayBuffer, DataView, JSON
  • 控制抽象对象 - Promise
  • 反射 - Reflect, Proxy
  • 国际化 - Intl
  • 非标准的对象 - Generator, Iterator, ParallelArray, StopIteration
