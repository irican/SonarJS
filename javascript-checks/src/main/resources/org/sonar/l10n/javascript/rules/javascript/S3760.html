算术表达式(/， *， %， ++，——，-，-=，*=，/=，%=，+=，+)，一元运算符(-)，或比较运算符( >， <， > =， < =)，其中一个或两个操作数是字符串，布尔值或日期值依赖于隐式转换。这样一段代码的可维护性和可靠性都值得怀疑。

不合规的代码示例

str = "80";
quarter = str / 4; // 不合规

if (str < 10) { // 不合规
   // ...
}

合规的解决方案

str = "80";
parsedStr = parseInt(str);
quarter = parsedStr / 4;

if (parsedStr < 10) {
  // ...
}

例外

  • 使用具有至少一个字符串操作数的二进制+操作符的表达式将被忽略，因为在这种情况下，+操作符将执行连接。
  • 如果两个操作数都是字符串，那么比较将被忽略，因为在这种情况下会执行词法比较。
