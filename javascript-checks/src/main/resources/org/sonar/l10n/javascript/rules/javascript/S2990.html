当关键字this在对象外部使用时，它指的是全局this对象，这与标准web页面中的window对象相同。这样的使用可能会让维护人员感到困惑。 相反，只需删除this，或替换为window;它将有同样的效果，更易于阅读。

不合规的代码示例

this.foo = 1;   // 不合规
console.log(this.foo); // 不合规

function MyObj() {
  this.foo = 1; // 合规
}

MyObj.func1 = function() {
  if (this.foo == 1) { // 合规
    // ...
  }
}

合规的解决方案

foo = 1;
console.log(foo);

function MyObj() {
  this.foo = 1;
}

MyObj.func1 = function() {
  if (this.foo == 1) {
    // ...
  }
}
