Web SQL数据库标准从未见过天日。它首先被制定出来，然后被W3C所反对，并且只在一些浏览器中实现。(在Firefox或IE中不支持)

此外，使用Web SQL数据库会带来安全问题，因为访问这样的数据库只需要它的名称。

不合规的代码示例

var db = window.openDatabase("myDb", "1.0", "Personal secrets stored here", 2*1024*1024);  // 不合规
