调试器语句可以放在过程中的任何位置以暂停执行。使用debugger语句类似于在代码中设置断点。 根据定义，必须从源代码中删除这样的语句，以防止在生产中出现任何意外行为或添加对攻击的漏洞。

不合规的代码示例

for (i = 1; i<5; i++) {
  // 将i打印到输出窗口。
  Debug.write("loop index is " + i);
  // 等待用户恢复。
  debugger;
}

合规的解决方案

for (i = 1; i<5; i++) {
  // 将i打印到输出窗口。
  Debug.write("loop index is " + i);
}
