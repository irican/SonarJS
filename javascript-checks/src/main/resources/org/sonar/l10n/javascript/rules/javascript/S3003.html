使用带字符串的比较运算符 (<, <=, >=, >) 不太可能产生预期的结果。确保目的是比较字符串而不是数字。

不合规的代码示例

var appleNumber = "123";
var orangeNumber = "45";
if (appleNumber < orangeNumber) {  // 不合规, 这个条件成立
  alert("There are more oranges");
}

合规的解决方案

var appleNumber = "123";
var orangeNumber = "45";
if (Number(appleNumber) < Number(orangeNumber)) {
  alert("There are more oranges");
}
