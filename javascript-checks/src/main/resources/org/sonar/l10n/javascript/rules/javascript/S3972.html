当每个语句都有自己的行时，代码最清楚。尽管如此，将if 及其结果 then 语句组合在同一行是一种常见的模式。 但是，当if与前一个else或else if的结束符} 放在同一行时， 它要么是一个错误—else丢失—要么是将来可能会出现错误，因为维护人员无法理解这两条语句是不连接的。

不合规的代码示例

if (condition1) {
  // ...
} if (condition2) {  // 不合规
  //...
}

合规的解决方案

if (condition1) {
  // ...
} else if (condition2) {
  //...
}

或

if (condition1) {
  // ...
}

if (condition2) {
  //...
}
