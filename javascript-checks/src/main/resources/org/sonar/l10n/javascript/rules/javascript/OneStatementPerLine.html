为了更好的可读性，不要在一行中放置多个语句。

不合规的代码示例

foo(); bar(); // 不合规

合规的解决方案

foo();
bar();

例外

包含一条语句的匿名函数将被忽略。带有单个嵌套语句的控制流语句也将被忽略。

onEvent(function() { doSomething(); });               // 合规
onEvent(function(p) { doSomething(); return p % 2; }); // 不合规

if (condition) doSomething();                         // 合规
if (condition) { doSomething(); }                     // 合规
