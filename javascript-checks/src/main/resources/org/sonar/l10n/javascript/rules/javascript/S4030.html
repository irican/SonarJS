当一个集合被填充但它的内容从未被使用时，那么它肯定是某种错误。要么重构使集合没有意义，要么缺少访问。

当集合上除了那些添加或删除值的方法外没有其他方法被调用时，此规则会引发一个问题。当集合上除了那些添加或删除值的方法外没有其他方法被调用时，此规则会引发一个问题。

不合规的代码示例

function getLength(a, b, c) {
  const strings = [];  // 不合规
  strings.push(a);
  strings.push(b);
  strings.push(c);

  return a.length + b.length + c.length;
}

合规的解决方案

function getLength(a, b, c) {
  return a.length + b.length + c.length;
}
