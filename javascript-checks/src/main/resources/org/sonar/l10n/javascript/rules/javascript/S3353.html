将初始化后未更改的变量const 标记为 "不，这个没有更新，也不应该更新". 为了代码的利益，应该在这些情况下使用代码const 。

不合规的代码示例

function seek(input) {
  let target = 32;  // 不合规
  for (let i of input) {
    if (i == target) {
      return true;
    }
  }
  return false;
}

合规的解决方案

function seek(input) {
  const target = 32;
  for (let i of input) {
    if (i == target) {
      return true;
    }
  }
  return false;
}
