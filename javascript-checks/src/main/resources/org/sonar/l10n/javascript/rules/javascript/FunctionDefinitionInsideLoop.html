在循环中定义一个函数可能会产生意想不到的结果。这样的函数保持对外部范围中定义的变量的引用。 因此，在循环中创建的所有函数实例都可以看到这些变量的相同值，这可能不是期望的。

不合规的代码示例

var funs = [];
for (var i = 0; i < 13; i++) {
  funs[i] = function() { // 不合规
    return i;
  };
}
console.log(funs[0]()); // 13 instead of 0
console.log(funs[1]()); // 13 instead of 1
console.log(funs[2]()); // 13 instead of 2
console.log(funs[3]()); // 13 instead of 3
