会话存储和本地存储是HTML 5的特性，它允许开发人员轻松地在客户端存储兆字节的数据，而不是4Kb的cookie。虽然以这种方式存储敏感信息有助于加快客户端应用程序的速度，但以这种方式存储敏感信息可能是危险的， 因为默认情况下数据没有加密，页面上的任何脚本都可能访问它。

当使用localStorage和sessionStorage API时，该规则会引发一个问题。

不合规的代码示例

localStorage.setItem("login", login); // 不合规
sessionStorage.setItem("sessionId", sessionId); // 不合规
