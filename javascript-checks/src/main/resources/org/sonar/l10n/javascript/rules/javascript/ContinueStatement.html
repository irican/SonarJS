continue 是一个非结构化的控制流语句。它降低了代码的可测试性、可读性和可维护性。应该使用诸如 if 之类的结构化控制流语句。

不合规的代码示例

  for (i = 0; i < 10; i++) {
    if (i == 5) {
      continue;  /* 不合规 */
    }
    alert("i = " + i);
  }

合规的解决方案

  for (i = 0; i < 10; i++) {
    if (i != 5) {  /* 合规 */
      alert("i = " + i);
    }
  }

请参阅

  • MISRA C:2004, 14.5 - The continue statement shall not be used.
