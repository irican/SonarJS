使用带有两个数字的+，您将得到加法。但是将它与字符串和其他任何东西一起使用，就会得到串联。这可能会令人困惑，特别是如果其中一个操作数不是字符串的话。建议显式地转换非字符串组件，以便将来的维护者更容易理解。

当+与字符串和非字符串一起使用时，该规则会引发一个问题。

不合规的代码示例

function foo() {
  let x = 5 + 8;  // okay
  let z = "8"
  return x + z;  // 不合规; yields string "138"
}

合规的解决方案 

function foo() {
  let x = 5 + 8;
  let z = "8"
  return x + Number(z);
}
