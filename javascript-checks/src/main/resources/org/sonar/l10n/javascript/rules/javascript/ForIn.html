for...in语句允许您循环遍历对象的所有属性的名称。属性列表包括通过原型链继承的所有属性。 当您对数据属性感兴趣时，这就产生了提供函数的副作用。不考虑这一点的程序可能会失败。

因此，每个 for...in 语句的主体都应该封装在一个 if语句中，该语句过滤作用于哪些属性。 它可以为特定类型或值范围进行选择，也可以排除函数，或者排除原型中的属性。

不合规的代码示例

for (name in object) {
    doSomething(name);  // 不合规
}

合规的解决方案

for (name in object) {
  if (object.hasOwnProperty(name)) {
    doSomething(name);
  }
}

例外

用于克隆对象的循环将被忽略。

for (prop in obj) {
  a[prop] = obj[prop];  // 合规的例外情况
}
