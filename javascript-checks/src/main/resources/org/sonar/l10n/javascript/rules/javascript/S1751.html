在循环中有一个无条件 break, return 或 throw会使其失效;循环只执行一次， 循环结构本身只是浪费按键。

循环中使用无条件的continue 本身就是浪费按键

由于这些原因，除了函数或方法中的最终 return 外，不应该使用无条件跳转语句。

不合规的代码示例

for (i = 0; i < 10; i++) {
  console.log("i is " + i);
  break;  // 循环只执行一次
}

for (i = 0; i < 10; i++) {
  console.log("i is " + i);
  continue;  // 这是毫无意义的;这个循环无论如何都会继续下去
}

for (i = 0; i < 10; i++) {
  console.log("i is " + i);
  return;  // 循环只执行一次
}

合规的解决方案

for (i = 0; i < 10; i++) {
  console.log("i is " + i);
}
