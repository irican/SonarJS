除了从语法角度看比较迟钝之外，函数构造函数也是危险的:它们的执行评估构造函数的字符串参数的方式类似于eval的工作方式，这可能会将您的程序暴露给随机的、非预期的代码，这些代码可能既慢又有安全风险。

一般来说，最好完全避免使用它，特别是在解析JSON数据时。您应该使用ECMAScript 5的内置JSON函数或专用库。

不合规的代码示例

var obj =  new Function("return " + data)();  // 不合规

合规的解决方案

var obj = JSON.parse(data);

例外

如果参数是字符串文字(例如(Function('return this'))())，函数调用将被忽略。
